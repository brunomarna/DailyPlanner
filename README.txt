# Daily Planner
A personalized, glassmorphism-style web dashboard designed to manage daily home tasks, medications, and bills with a touch of fun and creativity.

## 🔑 Key Features
### 📝Task Manager
* Efficient checklist to keep your daily chores organized.

### 🕐Alarm for Medication & Tasks
* Integrated reminders to ensure you never miss a dose or a deadline.

### 💊 Medical Cabinet 
*Daily Use (Auto-Deduct):** Automatically subtracts your daily dosage from the inventory.
*Non-Daily (Manual):** Keeps track of occasional medications.
*Visual Inventory Alert:** A dashboard warning section that highlights when stock is low (1 month or empty).
*Strike-through Logic:** Items are visually crossed out when they reach 0 months.

### 💷 Bill Calculator & tracker
* Manage bills and monthly payments with ease.

### ⏲️ Kitchen
* Featuring custom "Stray Chick" animations and sounds.

### 🛒 Groceries List
* Quick-add list for essential household items with autosave feature.

### 🖊️ Notes
* **Notes:** A quick space for thoughts.

### 🏖️ Relax Corner
* A dedicated space for a mental break.

### 💭 Dream & Gift Ideas Board
* Track things you want or gifts for others.

## 🛠️ Tech Stack
*Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript (ES6+)
*Backend:** Firebase (Firestore & Authentication)
*Hosting:** Firebase Hosting

## 🔐 Security & Setup
This project uses **Firebase Security Rules** to ensure that users can only read and write their own data.

### 💻 Local Setup
1. Clone the repository.
2. Create a `firebase-config.js` file in the root directory using the `firebase-config-template.js` as a guide.
3. Add your own Firebase API keys.
4. Run locally using a Live Server.
